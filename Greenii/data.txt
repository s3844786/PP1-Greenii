The database type is Mysql, and the version is Mysql 8.0.24. The database system is made using the hibernate method, and a table is automatically created when it is run for the first time. The database will run locally, and the relevant credential information is in application.properties. There are 4 tables: orders table, roles table, user table and user_role table. The user_role table is a join table for a many-to-many relationship between users and roles, and it consists of user_id and role_id attributes.
